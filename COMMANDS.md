# Telegram-бот — Все команды и функционал

## 🔑 Текстовые команды (ответом на сообщение)

Просто ответь на сообщение участника нужным словом:

| Слово | Действие | Нужен ранг |
|-------|----------|------------|
| **Повысить** | Повысить до следующего ранга (1→2→3) | 👑 Главный (3) |
| **Понизить** | Снять админа бота | 👑 Главный (3) |
| **Пред** | Выдать предупреждение | 🔰 Младший (1) |
| **Мут** | Замутить на 1 час | 🔰 Младший (1) |
| **Бан** | Забанить в группе | 🛡️ Администратор (2) |

---

## ⚙️ Управление админами

| Команда | Описание | Нужен ранг |
|---------|----------|------------|
| `/addadmin @user [1-3]` | Назначить админа с рангом | 👑 Главный (3) |
| `/removeadmin` | Снять админа (ответом) | 👑 Главный (3) |
| `/adminlist` | Список админов бота | Любой |
| `/admin` | Панель управления чатом | 👑 Главный (3) |

### Ранги админов

| Ранг | Emoji | Права |
|------|-------|-------|
| 1 | 🔰 Младший | Пред, Мут |
| 2 | 🛡️ Администратор | + Бан, заметки, стикеры |
| 3 | 👑 Главный | + Назначение админов, /admin панель |

## ⚙️ Требования

- Бот должен быть **администратором** группы в Telegram
- Для `/ban`, `/zombies` — право **Ban users**
- Для `/purge`, `/clean` — право **Delete messages**
- For `/pin` — право **Pin messages**
- Для `/mute` — право **Restrict members**
- Для `/setlog` — бот должен быть админом канала

---

## 🛡️ Автоматическая защита

Бот автоматически проверяет сообщения и блокирует:
- **Спам** — повторяющиеся сообщения
- **Мат** — нецензурную лексику (с учётом замены букв)
- **Ссылки** — если включён фильтр ссылок
- **Медиа** — если включён фильтр медиа
- **NSFW** — порнографический контент и стикеры
- **Пересылки** — пересланные сообщения из каналов
- **Телефоны/email** — контактные данные
- **Капча** — для новых участников (если включена)

При нарушении: удаление сообщения + предупреждение. При 3 предупреждениях — автоматический мут.
