# BosskoBroBot — Все команды

## 🛡️ Модерация (админы)

| Команда | Описание |
|---------|----------|
| `/warn` | Предупредить (ответом) |
| `/unwarn` | Снять предупреждение (ответом) |
| `/mute [время]` | Замутить (ответом, время: 1h, 30m, 5m) |
| `/unmute` | Размутить (ответом) |
| `/warnings` | Список предупреждений (ответом) |
| `/ban` | Забанить (ответом или по ID) |
| `/purge [N]` | Удалить N сообщений (ответом) |
| `/mutelist` | Список замученных |
| `/clean` | Удалить все сообщения бота |

## ⚙️ Управление чатом (админы)

| Команда | Описание |
|---------|----------|
| `/admin` | Панель управления (вкл/выкл защиты) |
| `/setrules <текст>` | Установить правила |
| `/delrules` | Удалить правила |
| `/slowmode [сек]` | Медленный режим (0 = выкл) |
| `/setlog` | Канал для логов (переслать из канала) |
| `/remlog` | Убрать канал логов |
| `/pin` | Закрепить сообщение (ответом) |
| `/unpin` / `/unpin all` | Открепить |
| `/allowlink <домен>` | Добавить ссылку в белый список |
| `/zombies` | Удалить мёртвые аккаунты |

## ⚙️ Требования
- Бот должен быть **администратором** в группе
- Для `/zombies`, `/mute`, `/ban` нужно право **ban users**
- Для `/purge`, `/clean` нужно право **delete messages**
- Для `/pin` нужно право **pin messages**
- Для `/setlog` бот должен быть админом в канале
